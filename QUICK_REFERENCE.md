## 🚀 快速启动

### Docker 方式（推荐）
```bash
cd docker
docker-compose up -d
```

### 本地开发

**后端：**
```bash
cd backend
mvn spring-boot:run
```

**前端：**
```bash
cd frontend
npm install
npm run dev
```
